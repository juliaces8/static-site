# Instrument World Store
### Static Website Deployment - Software Engineering Task

A responsive, modern e-commerce landing page for a premium musical 
instrument store. This project demonstrates front-end layout skills 
and automated deployment via GitHub Pages.

---

## 🚀 Key Features

* **Responsive Design:** Built with Bootstrap 5 to ensure a seamless 
  experience across mobile, tablet, and desktop devices.
* **Interactive UI:** Features custom CSS transitions, including 
  elevating product cards and image scaling on hover.
* **Modern Grid:** Implements a dynamic 5-column layout for product 
  showcasing.
* **Contact Integration:** Includes a professional footer with a 
  functional UI for customer inquiries.
